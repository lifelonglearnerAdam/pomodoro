# 🍅 番茄钟（Pomodoro Timer）

一个简洁优雅的网页番茄钟，帮助你在专注与休息之间保持最佳节奏。

## ✨ 功能

- **25 分钟专注 + 5 分钟休息**：经典的番茄工作法节奏
- **多个预设时长**：15 / 25 / 45 / 60 分钟，一键切换
- **开始 / 暂停 / 重置**：完整的计时控制
- **环形进度条**：直观展示剩余时间
- **自动模式切换**：专注结束后自动进入休息，休息结束后自动回到专注
- **番茄计数**：记录已完成的专注轮数

## 🚀 使用方式

直接打开 **在线版**：

👉 [lifelonglearneradam.github.io/pomodoro](https://lifelonglearneradam.github.io/pomodoro/)

或者下载 `index.html`，用浏览器打开即可，无需安装任何依赖。
